# OpenCode API Model Config

Manage OpenCode model API configurations, including adding, updating, and modifying providers and models.

## Install

```bash
npx skills add owofile/opencode-api-model-config
```

## Features

- Add new providers and models
- Update existing model lists
- Modify API keys
- Delete providers
- Rename providers
- Built-in templates for domestic providers (Zhipu AI, DeepSeek, SiliconFlow, Moonshot)

## When to Use

This skill activates when the user mentions:
- "add API", "configure API", "add model", "add provider"
- "update model list", "add new model"
- "change API key", "update key", "switch API"
- "view current config", "list existing models"
- "how to add API for xx model", "I want to use xx model"
- proxy/gateway API configuration
- any OpenCode model configuration needs
