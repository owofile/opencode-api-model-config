# OpenCode API Model Config

管理 OpenCode 的模型 API 配置，包括添加、更新、修改 provider 和模型。

## 安装

```bash
npx skills add NANAFREE/opencode-api-model-config
```

## 功能

- 添加新的 provider 和模型
- 更新现有模型列表
- 修改 API Key
- 删除 provider
- 重命名 provider
- 内置国内常用 provider 模板（智谱 AI、DeepSeek、硅基流动、Moonshot）

## 使用场景

当用户提到以下情况时使用此技能：
- "添加API"、"配置API"、"添加模型"、"添加provider"
- "更新模型列表"、"添加新模型"
- "修改API key"、"更新key"、"更换API"
- "查看当前配置"、"查看已有模型"
- "xx模型的API怎么添加"、"我想用xx模型"
- 中转API、代理API配置
- 任何关于 opencode 模型的配置需求
